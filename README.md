## 📌 Descripción

Este repositorio actúa como anexo digital y contiene todo el material técnico, gráfico y de software generado durante el desarrollo del Trabajo de Fin de Grado. 

---

## 📂 Estructura de Directorios

El material se encuentra clasificado en las siguientes carpetas para facilitar su revisión:

* **`01_Codigo_Python/`**
  Contiene la aplicación con interfaz gráfica (`app_alicona.py`) desarrollada para la automatización, extracción masiva y volcado de los datos metrológicos generados por los equipos ópticos.
* **`02_Planos/`**
  Modelos CAD y planimetría de detalle de las probetas de ensayo empleadas para el Niobio C103 y el Inconel 718.
* **`03_Estrategias_CAM/`**
  Archivos de simulación y trayectorias de fresado en 5 ejes programadas para abordar las irregularidades del material depositado por impresión 3D.
* **`04_Metrologia_Alicona/`**
  Matrices de datos en bruto, informes de desgaste de herramienta y perfiles topográficos tridimensionales capturados para determinar la calidad superficial previa a la aplicación de recubrimientos protectores.
* **`05_Catálogos/`**
  Catálogos de las herramientas de corte utilizadas.
* **`05_Normas_metrología/`**
  Normas aplicadas para la medición de los parametros de metrología.
